# AI Vision Architecture

A cutting-edge computer vision pipeline utilizing local multi-modal LLMs (llama3.2-vision) to parse, understand, and extract structural data from complex visual inputs, user interfaces, and architectural diagrams.

## Key Features
* Real-time inference running completely locally on RTX 4080 (16GB VRAM)
* Translates raw UI screenshots into structural React component code
* Integrated via API for automated n8n workflow processing
* Agnostic vision-to-text pipeline with deterministic JSON outputs

**Tech Stack:** Python, PyTorch, Ollama, Multi-Modal, Computer Vision
