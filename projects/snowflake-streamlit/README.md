# Snowflake AI Data App

![Snowflake Streamlit Dashboard](https://images.unsplash.com/photo-1460925895917-afdab827c52f?auto=format&fit=crop&w=1200&q=80)

An interactive Streamlit application deployed directly within Snowflake. It leverages Snowpark for high-performance data processing and integrates Cortex AI for natural language querying over massive enterprise datasets.

## Key Features
* Native deployment within Snowflake using Streamlit in Snowflake (SiS)
* Zero data movement architecture - compute is brought to the data
* Natural Language to SQL utilizing Snowflake Cortex LLMs
* Interactive forecasting and anomaly detection visualizations

**Tech Stack:** Python, Streamlit, Snowflake, Snowpark, AI/ML
